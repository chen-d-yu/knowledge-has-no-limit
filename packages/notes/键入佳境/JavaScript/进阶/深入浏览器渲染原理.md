# 浏览器渲染原理

## 前言

在前端的面试中，有一道经典的八股文问题，当我们在地址栏输入URL后，这个过程会发生什么？

这篇文章就来了解一下这背后的发生了什么

关于这个答案，一个最极简的回答是：

>当我们从一个特定的网站请求网页时，浏览器从网络服务器获取特定的资源，然后在我们的客户端上显式该网页

实际上，当用户在地址栏内输入`URL`按下回车后，浏览器大致会发生以下步骤：`导航 -> 获取数据 -> 解析数据 -> 浏览器渲染`等步骤

## 导航

这个阶段是网页加载的第一步，指当用户点击链接、在浏览器地址栏输入链接、提交一个表单等方式请求一个网页时发生的过程
