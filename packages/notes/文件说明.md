第一层文件夹表示一个nav

nav下的每一个文件夹，表示一个sidebar

sidebar下可以嵌套二级目录

blog博客sidebar设置如下：
```ts
export const sidebar: DefaultTheme.Config['sidebar'] = {
  '/pit/': [
    {
      text: '踩坑记录',
      items: [
        { text: 'npm 踩坑记录', link: '/pit/npm' },
        { text: 'PC 踩坑记录', link: '/pit/pc' },
        {
          text: 'H5 踩坑记录',
          items: [
            { text: 'xx文件', link: '/pit/a' },
            { text: 'xx文件', link: '/pit/b' }
          ]
        }
      ]
    }
  ]
}
```


部署博客命令：sh deploy.sh
